# File and Directory Commands

## mkdir

Purpose:
To create a new folder.

Syntax:
mkdir <folder_name>

Example:
mkdir Devops

Output:
Devops folder create avuthundi.

---

## mkdir -p

Purpose:
Nested directories ni okesari create cheyyadaniki.

Syntax:
mkdir -p <parent>/<child>/<subchild>

Example:
mkdir -p Devops/Data/Fun/Hi

Output:
Devops, Data, Fun, Hi directories create avuthayi.

---

## touch

Purpose:
New empty file create cheyyadaniki.

Syntax:
touch <file_name>

Example:
touch notes.txt

Output:
notes.txt file create avuthundi.

---

## cp

Purpose:
File ni copy cheyyadaniki.

Syntax:
cp <source_file> <destination_file>

Example:
cp file1.txt file2.txt

Output:
file1.txt content file2.txt lo copy avuthundi.

---

## cp -r

Purpose:
Complete folder ni contents tho saha copy cheyyadaniki.

Syntax:
cp -r <source_folder> <destination_folder>

Example:
cp -r Data1 Data2

Output:
Data1 folder Data2 peru tho copy avuthundi.

---

## mv

Purpose:
File or folder ni move cheyyadaniki leda rename cheyyadaniki.

Syntax:
mv <source> <destination>

Example:
mv old.txt new.txt

Output:
old.txt file name new.txt ga maruthundi.

---

## rm

Purpose:
File ni delete cheyyadaniki.

Syntax:
rm <file_name>

Example:
rm file.txt

Output:
file.txt delete avuthundi.

---

## rm -r

Purpose:
Folder ni andhulo unna files and subfolders tho saha delete cheyyadaniki.

Syntax:
rm -r <folder_name>

Example:
rm -r Devops

Output:
Devops folder andhulo unna files/folders delete avuthayi.

---

## rm -rf

Purpose:
Folder ni force ga confirmation lekunda delete cheyyadaniki.

Syntax:
rm -rf <folder_name>

Example:
rm -rf Devops

Output:
Devops folder warning lekunda delete avuthundi.
