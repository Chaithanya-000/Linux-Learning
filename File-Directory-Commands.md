# File and Directory Commands

## mkdir
Purpose: To create a new Folder

Example:
mkdir Devops

## mkdir -p
Purpose: Make Nested(More) no.of Folders inside at one by one.

Example:
mkdir Devops/Data/Fun/Hi

## touch
Purpose: New file create cheyyadaniki

Example:
touch notes.txt

## cp
Purpose: File copy cheyyadaniki

Example:
cp file1.txt file2.txt

## cp -r
Purpose: Complete folder ni copy cheyadaniki

Example:
cp Data1

## mv
Purpose: File move/rename cheyyadaniki

Example:
mv old.txt new.txt

## rm
Purpose: File delete cheyyadaniki

Example:
rm file.txt

## rm -r
Purpose: To remove complete folder with exixting files also will be deleted

Example:
rm -r file.txt

## rm -rf
Purpose: Force Remove 

Example:
rm file.txt
