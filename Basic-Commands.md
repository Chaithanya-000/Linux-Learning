# Basic Commands

## pwd

Purpose:
Current working directory path chupisthundi.

Syntax:
pwd

Example:
pwd

Output:
/home/user/Documents laanti current directory path display avuthundi.

---

## ls

Purpose:
Current directory lo unna files and folders list chupisthundi.

Syntax:
ls

Example:
ls

Output:
Current directory contents display avuthayi.

---

## cd

Purpose:
Oka directory nundi vere directory ki move avvadaniki.

Syntax:
cd <directory_name>

Example:
cd Documents

Output:
Documents directory loki move avutharu.

---

## ls -l

Purpose:
Files and folders detailed information (permissions, owner, size, date) chupisthundi.

Syntax:
ls -l

Example:
ls -l

Output:
Detailed file and directory information display avuthundi.

---

## ls -la

Purpose:
Detailed information tho patu hidden files ni kuda chupisthundi.

Syntax:
ls -la

Example:
ls -la

Output:
Permissions, owner details mariyu hidden files display avuthayi.

---

## ls -a

Purpose:
Hidden files and folders ni chupisthundi.

Syntax:
ls -a

Example:
ls -a

Output:
.bashrc, .profile laanti hidden files display avuthayi.

---

## cd ..

Purpose:
Current directory nundi parent directory ki move avvadaniki.

Syntax:
cd ..

Example:
cd ..

Output:
One level back (parent) directory ki move avutharu.
