# Basic Commands

## pwd
Purpose: Current directory chupisthundi

Example:
pwd

## ls
Purpose: Files and folders list chupisthundi

Example:
ls

## cd 
Purpose: Directory change or Directory Backward

Example:
cd Documents

## ls -l
Purpose: List lo vunna files annitiki permissions tho saha chupistundhi

Example:
ls -l 

## ls -la
Purpose: List lo vunna files annitiki permissions tho patuuu hidden files ki kuda chupistundhi

Example:
ls -la

## ls -a
Purpose: List lo vunna hidden files ni chupistundhi

Example:
ls -a

## cd..
Purpose: Back to the Directory
