# Git and GitHub Commands

## git init

Purpose:
Current folder ni Git repository ga initialize cheyyadaniki.

Syntax:
git init

Example:
git init

Output:
Empty Git repository create avuthundi.

---

## git status

Purpose:
Current repository status chudadaniki.

Syntax:
git status

Example:
git status

Output:
Modified and untracked files display avuthayi.

---

## git add

Purpose:
Files ni staging area ki add cheyyadaniki.

Syntax:
git add <file_name>

Example:
git add notes.txt

Output:
File staging area ki add avuthundi.

---

## git commit

Purpose:
Changes ni save cheyyadaniki.

Syntax:
git commit -m "message"

Example:
git commit -m "Added Linux notes"

Output:
Changes repository lo save avuthayi.

---

## git log

Purpose:
Commit history chudadaniki.

Syntax:
git log

Example:
git log

Output:
Commit history display avuthundi.

---

## git clone

Purpose:
GitHub repository copy cheyyadaniki.

Syntax:
git clone <repository_url>

Example:
git clone https://github.com/user/repo.git

Output:
Repository local machine lo download avuthundi.

---

## git push

Purpose:
Local changes GitHub ki upload cheyyadaniki.

Syntax:
git push

Example:
git push origin main

Output:
Changes GitHub lo update avuthayi.

---

## git pull

Purpose:
GitHub nundi latest changes download cheyyadaniki.

Syntax:
git pull

Example:
git pull origin main

Output:
Latest changes local machine ki vastayi.
