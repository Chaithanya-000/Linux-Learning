# Disk and System Information Commands

## df

Purpose:
Disk space usage ni chupisthundi.

Syntax:
df

Example:
df

Output:
Mounted file systems disk usage display avuthundi.

---

## df -h

Purpose:
Disk space ni human-readable format lo chupisthundi.

Syntax:
df -h

Example:
df -h

Output:
Disk usage GB/MB format lo display avuthundi.

---

## du

Purpose:
Files and folders size chupisthundi.

Syntax:
du <folder_name>

Example:
du Documents

Output:
Documents folder size display avuthundi.

---

## du -sh

Purpose:
Folder total size ni human-readable format lo chupisthundi.

Syntax:
du -sh <folder_name>

Example:
du -sh Documents

Output:
Documents folder total size display avuthundi.
